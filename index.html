News Hack

A New York Times and USA Today news app.

  • iOS code
  • Android code

All the news you need.

News Hack lets users read popular news papers like The New York Times and USA Today for free.

Tech

Implements The New York Times API to pull article titles and urls.
Implements US Today API to pull article titles and urls.

Displays article data through Master-Detail application.
Does NSString manipulation in order to access restricted content.
Stores NSUserDefaults so users can choose what news sources are important to them so that only they get loaded when app is launched.
Uses size classes to look great on all iPhones.

Available for iOS
and Android
